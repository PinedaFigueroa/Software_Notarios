Bufete
======

En esta tabla se incluye todo lo que identifica a cada Bufete dentro del Sistema

Campos
------

- id: Integer
- nombre: String
- direccion_fisica: String
- telefono_contacto: String
- persona_autorizada: String
- nit: String
- correo_electronico: String
- forma_contacto_preferida: String
- datos_facturacion: String

Relaciones
----------

- usuarios
- notarios
- procuradores
- asistentes
- clientes
